ParSuit Python Application

This is a Python Application which allows users to set custom weights to conduct a Suitability Analysis in the Capitol Region of Connecticut. The application uses a variety of data sources to conduct the analysis and provides a final map of the most suitable areas for a user's needs in a interactive map.
