NYC Broadband Coverage Analysis

A comprehensive geospatial analysis project examining broadband internet coverage and demographics across New York City. This project integrates data from multiple authoritative sources including NYC Open Data, the US Census Bureau, and FCC broadband availability data.

The project features a complete ETL pipeline built with Python, spatial database management using PostgreSQL with PostGIS extension, and sophisticated SQL-based market analysis. Geographic data processing covers NYC boroughs and census block groups, enriched with demographic information to provide comprehensive coverage insights.

Key technologies include PostgreSQL/PostGIS for spatial database operations, Python for data processing and API integration (Census Bureau API), and QGIS for professional cartographic visualization. The project demonstrates advanced GIS development skills including spatial joins, geometric operations, and multi-source data integration.
